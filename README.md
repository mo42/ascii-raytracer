# A Real-time ASCII Raytracer in your Terminal

I was curious if I one can do real-time ASCII raytracing in the terminal.

![Real-time ASCII Raytracer in the terminal](.example.mp4)

## Compilation
```sh
git clone https://github.com/mo42/ascii-raytracer.git
cd ascii-raytracer
mkdir build
cd build
cmake ..
make
```
